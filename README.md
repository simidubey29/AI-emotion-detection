# AI Advanced Emotion Detection System

## 📌 Project Overview
AI Advanced Emotion Detection System is a real-time computer vision project that detects human emotions using webcam video feed.

The system uses:
- OpenCV for webcam and face detection
- DeepFace for AI emotion analysis
- TensorFlow backend for deep learning
- Python for implementation

The project can detect:
- Happy
- Smile
- Sad
- Angry
- Fear
- Surprise
- Shock
- Neutral
- Disgust

It also includes:
- FPS Counter
- Stable Emotion Prediction
- Snapshot Saving
- Emotion Statistics
- Real-Time Face Detection
